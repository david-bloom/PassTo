## Product Goal

Give the nurse a launch-ready dashboard showing credential, license, wallet, and subscription status, with only launch-critical actions exposed.

## Technical Scope

- Display credential status, license status, current-as-of timestamp, wallet readiness, and subscription/tier state.
- Expose launch-critical share-link action.
- Hide or disable deferred Show QR and PDF export launch blockers.
- Use server-side entitlement state as source of truth.
- Avoid client-only trust or entitlement decisions.

## Out of Scope

- Show QR implementation.
- PDF export implementation.
- Lovable admin UI.
- Deep analytics.

## Acceptance Criteria

- [ ] Nurse sees credential/license/wallet/subscription status from canonical Supabase data.
- [ ] Dashboard does not expose Show QR or PDF export as launch-critical actions.
- [ ] Share-link entry point is present for entitled users or correctly gated.
- [ ] Empty/error/loading states are handled.
- [ ] No privileged writes happen directly from dashboard UI.
- [ ] Claude documents tests, files changed, deviations, and risks.

## QA Plan

- Test dashboard states for no credential, pending credential, issued credential, wallet failure, and lapsed subscription.
- Verify UI reflects server-side data.
- Verify deferred actions are not launch blockers.
