## Product Goal

Ensure PassTo password reset emails send nurses to the live PassTo enrollment domain instead of `http://localhost:3000/`.

David reported on 2026-06-02 that a password reset email was received, but its link points to `http://localhost:3000/`. This breaks launch-critical account recovery and indicates Supabase Auth URL configuration and/or Lovable reset redirect behavior is still using a local development URL.
